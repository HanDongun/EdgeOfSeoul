# EdgeOfSeoul

## 프로젝트 여기(여행과 기록)
 
여행 기록을 일지별로 제공하고 사진의 위도, 경도를 기반으로 지도 정보를 제공합니다.

관광지 api를 활용하여 위도, 경도 기반으로 관광지의 정보를 제공합니다.

    * 피드 별 지도
