# EdgeOfSeoul

## 프로젝트 여기(여행과 기록)

여기 프로젝트는 사용자에게 지도 기반의 여행 일지 작성과 관광지 검색 기능을 제공하는 모바일 어플리케이션입니다.


### 프로젝트 구성

* 로그인
  * 회원가입
  * 비밀번호 찾기
* 프로필 화면
  * 일지 보기
    * 일지 별 지도
* 일지 추가
  * 일지 내용 작성
    * 일지 지도 보기
* 관광지 검색
  * 관광지 사진 보기
